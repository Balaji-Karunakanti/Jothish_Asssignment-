A modern ReactJS application built as part of an internship assignment.
The application includes:

🔐 Login Page

📋 Employee List Page

👤 Employee Details Page

📷 Camera Capture Feature

🎨 Consistent Modern UI Design

🚀 Features
🔐 Login Page

Role-based login

School/organization selection

Remember Me functionality

Responsive UI with consistent color palette

Redirects to List Page on successful login

📋 List Page

Displays employee data in a structured format

Clean table layout

Clickable rows

Navigation to Details Page

Search / filter ready structure

👤 Details Page

Displays complete employee information

Modern card-based UI

📷 Capture photo using device camera

Live camera preview

Capture & display image inside the app

🛠️ Tech Stack

⚛️ ReactJS

🧭 React Router DOM

🎨 Tailwind CSS / Custom CSS

📷 Web Camera API (navigator.mediaDevices)
